# GeoAI Workshop: Mapping Rice Prices in Ghana

This repository contains code and teaching resources for a 2-hour hands-on GeoAI workshop. Participants will use Ghanaian market data to explore spatial affordability patterns for rice.

## Learning Outcomes
- Understand what GeoAI means in a practical Ghanaian context
- Load and inspect real-world food price datasets
- Convert tabular market locations into geospatial data
- Spatially join points to Ghanaâ€™s administrative boundaries
- Create choropleth maps of rice prices across regions
- Extend from regional to district-level analysis
- Optional: Ask natural language questions using an LLM

## Tools
- Python 3.10+
- GeoPandas
- Pandas
- Folium or Plotly Express
- Matplotlib
